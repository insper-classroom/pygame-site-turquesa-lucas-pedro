# ⚓ Captain Cannonball — Site Oficial

Bem-vindo ao porto digital de **Captain Cannonball**, um site de divulgação criado para apresentar um jogo de plataforma 2D com temática pirata, pixel art, ilhas perigosas, desafios de precisão e uma boa dose de aventura em alto-mar.

🌊 **Acesse a página publicada:**  
https://insper-classroom.github.io/pygame-site-turquesa-lucas-pedro/

---

## 🏴‍☠️ Sobre o Projeto

O site foi desenvolvido para divulgar o jogo **Captain Cannonball**, destacando sua proposta visual, mecânicas principais, mundo, screenshots, equipe, log de desenvolvimento, avaliação dos princípios CRAP e instruções de download/execução.

A página busca transmitir a atmosfera do jogo desde o primeiro contato: oceano, navios, tesouros, mapas antigos, dourado pirata e uma estética inspirada em games clássicos de plataforma.

---

## 🗺️ Seções do Site

- **Hero** — apresentação principal do jogo, CTA para download e link para o GitHub.
- **Sobre** — narrativa e pilares de gameplay.
- **Mundo** — vídeo de gameplay e descrição das regiões/fases.
- **Galeria** — capturas reais do jogo em três momentos da jornada.
- **Equipe** — apresentação dos desenvolvedores Lucas Ibanez e Pedro Zanin.
- **Log de Desenvolvimento** — registro temático das principais entregas do projeto.
- **Avaliação CRAP** — como o feedback de design ajudou a melhorar a versão final.
- **Download** — instruções para baixar, instalar dependências e executar o jogo.

---

## ✨ Recursos Implementados

- Layout responsivo para desktop e mobile.
- Visual temático com paleta naval, dourado, textura e atmosfera pirata.
- Hero com imagem de barco e camadas visuais.
- Galeria interativa com cards padronizados.
- Vídeo de gameplay integrado à seção Mundo.
- Cards com efeito hover/tilt.
- Animações de reveal no scroll.
- Cursor customizado temático.
- Ícone de guia em formato de âncora.
- Reset CSS separado em `docs/reset.css`.
- HTML semântico com `header`, `nav`, `main`, `section`, `article`, `figure`, `time` e `footer`.
